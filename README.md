# Explor

Explor is an app that provides an online canvas to upload your favorite trips.!!
# AAU 
* Should be able to add a blog
* Able to edit your blog
* Able to add photos of your blog
     
      

## Technologies Used

* JavaScript
* HTML5
* CSS3
* Python
* Heroku
* PostgreSQL
